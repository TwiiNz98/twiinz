Preguntas frecuentes

Todo lo que necesitas saber antes de empezar.

Precios y pagos

Los planes parten desde $75.000 CLP para una Landing Básica. Una Landing Starter parte desde $120.000 CLP, una Tienda Online Pro desde $160.000 CLP y el plan de Mantenimiento desde $40.000 CLP/mes. Siempre se define el precio exacto antes de empezar — nunca hay costos adicionales sorpresa.

Sí. El modelo estándar es 50% al iniciar y 50% al entregar. Para proyectos Pro o personalizados podemos coordinar un esquema de hasta 3 pagos.

Proceso y plazos

Depende del alcance. Una landing simple: 3–5 días. Un e-commerce completo: 10–15 días. Un sistema con base de datos y panel admin: 3–4 semanas. Siempre recibes una fecha de entrega exacta antes de empezar.

Cada plan tiene un número definido de rondas de revisión (1, 3 o ilimitadas). Una ronda equivale a una lista de cambios consolidada. Siempre apruebas el diseño antes de empezar el desarrollo.

No empezamos a construir sin que hayas aprobado el diseño. Si en las revisiones algo no te convence, lo ajustamos hasta que esté perfecto dentro del scope acordado.

Técnico

Sí, siempre. Al finalizar recibes el repositorio completo en GitHub. El código es tuyo. Sin suscripciones ni dependencias de mi parte para que tu sitio funcione.

No. Si el proyecto incluye panel admin, puedes gestionar todo el contenido desde una interfaz visual. Incluyo una sesión de onboarding para enseñarte a usarlo.

Depende del plan. Los proyectos con Firebase o panel admin sí permiten edición autónoma. Para proyectos estáticos, los cambios de contenido se coordinan por separado a tarifa preferencial.

Post-entrega

Claro. Los proyectos Pro incluyen soporte de 60 días. Para cualquier plan, las actualizaciones se presupuestan por separado a tarifa preferencial para clientes activos. También puedes contratar el plan de Mantenimiento para cambios continuos.

El plan Pro incluye hosting en Netlify (gratis para proyectos estáticos). El plan Mantenimiento incluye gestión completa de dominio y SSL. Los planes Básico y Starter no incluyen hosting pero te asesoro en la configuración.

Si el problema es por un error del código entregado, lo resuelvo sin costo adicional. Si el problema es por cambios en servicios de terceros (hosting, dominio), te asesoro en la resolución a tarifa preferencial.
